## 🚀 Getting Started

### Prerequisites
```bash
python >= 3.7
tensorflow >= 2.0
keras
```

### Installation
```bash
git clone https://github.com/Jerronce/neural-network-image-recognition.git
cd neural-network-image-recognition
pip install -r requirements.txt
```

### Quick Start
```bash
# Train the model
python train.py

# Evaluate performance
python evaluate.py

# Make predictions
python predict.py --image path/to/image.jpg
```
